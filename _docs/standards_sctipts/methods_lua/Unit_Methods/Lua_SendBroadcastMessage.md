# Lua SendBroadcastMessage

## Description

Sends a message to the chatbox of a single player.

## Usage/Example

```
pPlayer:SendBroadcastMessage("This is a broadcast Example for AE.")
```

```
pPlayer:SendBroadcastMessage("|cFFFFCC00Colored Example for AE|r")
```
