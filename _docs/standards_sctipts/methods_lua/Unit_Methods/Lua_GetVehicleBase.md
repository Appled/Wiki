# Lua GetVehicleBase

## Description

Returns the vehicle unit the unit is currently on.

## Syntax

```
GetVehicleBase(); -- No parameters
```

parameter(s)    
None    

return value(s)    
Returns a Unit on success.    
Returns nil on failure.
