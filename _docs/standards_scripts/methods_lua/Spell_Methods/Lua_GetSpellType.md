# Lua GetSpellType

## Description

Returns the type of the spell.

## Usage/Example

(../world/Spell/Definitions/SpellDamageType.h)

```
SPELL_DMG_TYPE_NONE   = 0
SPELL_DMG_TYPE_MAGIC  = 1
SPELL_DMG_TYPE_MELEE  = 2
SPELL_DMG_TYPE_RANGED = 3
```
