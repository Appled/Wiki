# Lua MoveVehiclePassengerToSeat

## Description

Moves a passenger to another seat.

## Syntax

```
MoveVehiclePassengerToSeat(unit, seat);
```

parameter(s)
unit - A passenger of the vehicle.    
seat - the seat where we want to move the passenger.    

return value(s)    
None
