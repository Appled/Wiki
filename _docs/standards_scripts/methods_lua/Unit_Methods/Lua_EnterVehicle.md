# Lua EnterVehicle

## Description

Makes the Unit enter a vehicle after a delay.

## Syntax

```
EnterVehicle(guid, delay);
```

parameter(s)    
guid  - guid of the vehicle to enter.    
delay - delay before entering in milliseconds.    

return value(s)    
None
