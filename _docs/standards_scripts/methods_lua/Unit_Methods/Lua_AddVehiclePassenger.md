# Lua AddVehiclePassenger

## Description

Spawns and adds a new NPC passenger to the vehicle.

## Syntax

```
AddVehiclePassenger(creature_entry);
```

parameter(s)    
creature_entry - The entry id of the NPC.    

return value(s)    
None
