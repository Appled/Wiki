# gossip_menu_option
This table contains the texts for menu options. 

## Structure

Field                                                                                    | Type    | Default | Comment
---------------------------------------------------------------------------------------- | ------- | ------- | -------
[entry](#entry)             | int(10) | AUTO    |        
[option_text](#option_text) | text    |         |        

### entry

The unique entry ID for this text.

### option_text

The option text.
